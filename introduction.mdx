  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Harness in minutes
  </p>

  <Steps>
    <Step title="Install Harness">
      Install via curl or cargo:

<CodeGroup>
```bash curl
curl -fsSL https://harness.lol/install.sh | sh
```

```bash cargo
cargo install harnesscli
```
</CodeGroup>
    </Step>

    <Step title="Verify installation">
      Check which agents are available on your system:

      ```bash
      harness list
      ```

      You'll see output like:

      ```text
      Available agents:
        - Claude Code (claude) [v1.2.3]
        - OpenCode (opencode) [v0.8.1]
      ```
    </Step>

    <Step title="Run your first task">
      Execute a prompt with your preferred agent:

      ```bash
      harness run --agent claude --prompt "explain this codebase"
      ```

      Harness spawns the agent and streams unified NDJSON events to stdout.
    </Step>

    <Step title="Configure your project">
      Create a `harness.toml` for project-level defaults:

      ```bash
      harness config init
      ```

      Then customize your settings:

      ```toml
      default_agent = "claude"
      default_model = "opus"
      default_permissions = "full-access"

      [agents.claude]
      model = "sonnet"
      extra_args = ["--verbose"]
      ```
    </Step>
  </Steps>
</div>
